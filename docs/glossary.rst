.. _glossary:

Glossary
========

.. glossary::

   AGDC
      The Australian Geoscience Data Cube

   API
      The Data Cube Application Programming Interface gives programmers full access to the capabilities
      of the Cube, allowing query and advanced data retrieval.

   NCI
      The Australian National Computational Infrastructure is Australia's national research computing facility. It
      provides computing facilities for use by Australian researchers, industry and government. For more information
      see http://www.nci.org.au/.

   Python
      The programming language used to develop the Data Cube. It is easy to use while still allowing high performance
      access and processing capabilities. See http://www.python.org/ for more.
